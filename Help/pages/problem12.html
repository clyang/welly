    Why I see some amazing characters which I cannot recognize when I'm querying an IP's actual address?
 

Basically, the characters you see is in simplified Chinese. We are sorry that we have no time to translate the IP-address database to languages other than simplified Chinese. We strongly recommend you learn some Chinese if you need to use this function frequently. We also welcome people who would like to help us port the database into other languages.
